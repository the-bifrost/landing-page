Protocols

ESP-NOW

A protocol developed by Espressif for direct device-to-device communication over Wi-Fi, without the need for routers. Ideal for fast mesh networks with low power consumption.

See more!

MQTT

A lightweight publish/subscribe protocol designed for IoT. It uses a broker to manage and route messages between connected devices efficiently.

See more!

LoRa

A long-range, low-power communication protocol perfect for remote areas. Commonly used in telemetry, environmental monitoring, and precision agriculture.

See more!

Zigbee

A short-range mesh network protocol based on the IEEE 802.15.4 standard. Widely used in smart home automation and energy-efficient smart devices.

See more!
